# 🍽️ Luxury Restaurant

A modern luxury restaurant website built with React, Vite, and Tailwind CSS.

## ✨ Features
- Responsive Design
- Elegant UI/UX
- Interactive Menu
- Reservation Form
- Chef Section
- Customer Testimonials
- Gallery
- Contact Section
- Smooth Animations

## 🛠️ Technologies
- React
- Vite
- Tailwind CSS
- Framer Motion
